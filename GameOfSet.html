A game where cards have symbols with different colors, shapes, numbers, and fillings. A set is three cards where every feature is either all the same, or all different. Grab as many sets as you can.

Hot keys: 'h' - Card Selection Hint: Automatically selects 2 of a set on the board if there is one available. If none, negative sound plays.
'n' - Number Hint: States how many sets are on the board.
's' - Statistics: Displays a board of a player's game statistics.
'd' - Draw Cards: Draws 3 new cards if there are no sets on the board. Negative sound plays if there is still a set.
Player 1 score: 0 Player 2 score: 0
Solo Player score: 0 Time remaining: 50 seconds
